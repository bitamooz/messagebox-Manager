راهنمای نصب و استفاده از MessageBox Manager

🔹 فایل اصلی پروژه را در ویژوال استودیو باز کنید

🔹 برای تست نمونه‌ها، فایل Form1.cs را بررسی نمایید

🔹 ۷ نوع مختلف MessageBox با کاربردهای متفاوت پیاده‌سازی شده است

🔹 هر MessageBox دارای مثال عملی و کدهای کامنت‌گذاری شده است

مراحل اجرای پروژه:

1. Clone the repository
git clone https://github.com/bitamooz/messagebox-Manager.git

2. Open in Visual Studio
3. Build the solution (Ctrl+Shift+B)
4. Run the project (F5)

🎯 نکات کلیدی:

- برای استفاده در پروژه‌های خود، کافیست متدهای مورد نیاز را کپی کنید

- می‌توانید پارامترها را براساس نیاز خود تغییر دهید
